Student Performance Analysis

This project analyzes student performance data using Python.

Contents
- student_analysis.ipynb: Jupyter Notebook containing
  - Data cleaning
  - Exploratory data analysis
  - Visualizations
  - Insights on study time, gender, and final grades

Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

Summary
The analysis explores how factors like study time and gender relate to students' final grades.
